Blackhole Simulator

A physics-based orbital mechanics simulator featuring various "satellites" like bananas, pizza, and spaghetti.

This project was created to explore the physics of black holes as a fun creative group assignment for a University Class.

How to Play

Use the mouse to interact with the simulation.

Click and drag to create objects with velocity.
